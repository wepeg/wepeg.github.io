Pengertian Link

HTML Link atau hyperlink,adalah sekumpulan kata ataupun gambar yang apabila di-klik akan menuju ke dokumen/halaman lain atau bagian lain dari dokumen/halaman yang sama

Macam-macam Link di dalam HTML:
  • Link Relatif
  • Link Absolute
  • Link Kebagian lain dalam Dokumen Yang sama
  • Link ke E-Mail

1. Link Relatif

Link Relatif dibuat apabila kita membuat suatu link pada page ke page yang lain pada komputer yang sama, tidak memerlukan alamat internet yang lengkap.Contoh kita akan membuka file hasil latihan ordered list kita minggu lain

Back to Top

2. Link Absolute

Link absolute dibuat apabila kita akan membuat link ke page web lain yang berada di luar website kita.Sehingga alamat yang kita tulis harus lengkap sesuai dengan alamat web yang akan kita tuju.Contoh kita akan membuat link ke web google.com

Back to Top

3. Link E-Mail

Link E-Mail digunakan untuk membuat link saat kita akan membuat/mengirim suatu e-mail(tidak bisa jika aplikasi email client tidak terinstal). contoh kita akan mengirim email ke alamatfelisyaus@gmail.com

Back to Top

4. Link Ke Bagian Lain di dalam Dokumen/Halaman Yang Sama

Link ini digunakan apabila page yang ada terlalu panjang sehingga seringkali harus melakukan scrool untuk menuju bagian yang diinginkan,untuk mempermudah kita gunakan link dokumen ke bagian lain pada halaman yang sama.yang harus dilakukan adalah:

  1. Membuat id pada bagian yang di tuju
  2. Mambuat script link pada tulisan yang kita inginkan

Misal kita akan membuat link pada dokumen latihan ini pada bagian atas yaitu jenis-jenis Link, silahkan pelajari script nya
